23. 选择器优先级

CSS选择器优先级从高到低：

  1. 内联样式（style属性）
  2. ID选择器（#id）
  3. 类选择器、属性选择器、伪类（.class, [attr], :hover）
  4. 元素选择器、伪元素（div, ::before）

说明：当多个规则应用到同一元素时，优先级高的规则会覆盖优先级低的规则。

24. 响应式设计（媒体查询）

媒体查询（@media）用于根据设备特性应用不同的样式。

  • max-width: 768px - 平板设备样式
  • max-width: 480px - 手机设备样式
  • print - 打印样式

说明：调整浏览器窗口大小查看响应式效果。在小屏幕上，布局会自动调整为单列。
