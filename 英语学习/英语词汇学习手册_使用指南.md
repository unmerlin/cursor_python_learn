## 💡 高效学习技巧

### 1. 结合场景记忆
✅ **编程时**：
```python
# 看到 variable（变量），想：var = 变化
user_name = "Alice"  # 用户名是变量，可以变化

# 看到 function（函数），想：func = 功能
def play_music():  # 函数执行播放音乐的功能
    pass
```

✅ **听歌时**：
- 听到 "love" → 联想：我love编程和音乐
- 听到 "dream" → 联想：我的coding dream
- 听到 "feel" → 联想：我feel到代码的rhythm

✅ **学音乐时** 🎵：
```
学scale(音阶) → 在钢琴上弹C major scale
学chord(和弦) → 用MIDI编程生成和弦
学tempo(速度) → 设置音乐app的BPM
学rhythm(节奏) → 创建drum loop编程练习
```

### 2. 词根记忆法
利用词根理解单词家族：
- **port（携带）**: import(导入), export(导出), portable(便携的)
- **struct（建造）**: structure(结构), construct(构建), instruct(指导)
- **scribe（写）**: describe(描述), subscribe(订阅), script(脚本)

### 3. 联想记忆法
创建有趣的联想：
- **debug** = de(去除) + bug(虫子) → 想象在代码里抓虫子
- **compile** = com(一起) + pile(堆) → 把代码堆在一起
- **melody** = melo(甜美) + dy → 甜美的旋律

### 4. 分类记忆法
按主题分组记忆：
- **情感词**: love, happy, sad, joy, pain
- **动作词**: play, sing, dance, code, debug
- **时间词**: day, night, moment, forever
- **数据类型**: string, integer, boolean, list
- **音乐术语**: scale, chord, rhythm, tempo, harmony 🎵

### 5. 造句记忆法
用新词造自己的句子：
- 学会 "loop" → 我造句：I loop through my favorite songs. (我循环播放我最喜欢的歌)
- 学会 "heart" → 我造句：Music touches my heart while coding. (编程时音乐触动我的心)

## ❓ 常见问题

### Q1: 每天学多少词合适？
**A**: 建议30-50词。质量比数量重要，确保真正掌握。

### Q2: 忘记的词怎么办？
**A**: 正常现象！使用间隔复习法：第2天、第7天、第30天各复习一次。

### Q3: 怎么判断是否真正掌握？
**A**: 能在实际场景中自然使用，不需要思考就能理解和使用。

### Q4: 需要全部背下来吗？
**A**: 不需要！重点是理解和应用。高频词要熟练，低频词能认识即可。
