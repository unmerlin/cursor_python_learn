### Step 3: 开始学习（今天就行动）

**学习一个单词的流程**（2-3分钟）：

```
以 "variable" 为例：

1️⃣ 看单词：variable
2️⃣ 读音标：/ˈveəriəbl/ (跟读3遍)
3️⃣ 看中文：变量
4️⃣ 理解词根：var(vary变化) + able(能够)
5️⃣ 读例句：x = 10  # x is a variable storing the value 10
6️⃣ 用记忆法：var表示变化，变量可以存储不同的值
7️⃣ 自己造句：song_name is a variable. (歌名是变量)
```

## 💡 学习小贴士

### ✅ 推荐做法
- 每天固定时间学习（建立习惯）
- 在实际场景中使用新词（编程注释、听歌识词）
- 用笔记本或App记录难词
- 大声朗读例句（提升语感）
- 每周末总复习

### ❌ 避免错误
- 不要贪多，每天30-50词足够
- 不要只看不用，要主动应用
- 不要跳过复习，遗忘很正常
- 不要死记硬背，理解词根更高效
- 不要孤立记忆，结合场景学习
